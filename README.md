# Cosme_VISION

Cosme_VISION은 화장품의 광고 문구가 아니라 `실제 성분`을 기준으로 제품을 분석하고 선택하기 위한 웹 플랫폼 프로젝트입니다.
```text
Powered by Dabin Jeon
```
이 프로젝트는 아주 단순한 질문에서 시작했습니다.

`이 화장품은 정말 내가 기대하는 기능을 할까?`

화장품을 고를 때 미백, 진정, 자외선 차단, 보습 같은 표현은 넘쳐나지만, 실제로 어떤 성분이 들어 있고 그 성분이 어떤 역할을 하는지까지 보고 선택하는 일은 쉽지 않습니다.  
저는 화장품을 사용할 때 광고보다는 성분을 기준으로 판단하고 싶었고, 내가 실제로 사용했을 때 피부에 맞았는지, 혹은 트러블이 생겼는지를 함께 분석할 수 있는 플랫폼이 필요하다고 느꼈습니다.

## 프로젝트를 시작한 이유

같은 기능을 내세우는 화장품이라도 실제 성분 구성은 크게 다를 수 있습니다.  
또 어떤 제품은 내 피부에 잘 맞지만, 어떤 제품은 비슷해 보여도 트러블이나 자극을 유발할 수 있습니다.

Cosme_VISION은 이런 질문에 답하기 위해 기획했습니다.

- 같은 기능 카테고리 안에서 어떤 제품이 실제로 어떤 성분을 가지고 있는가?
- 내가 사용했던 제품 중 피부 트러블을 일으킨 제품들은 어떤 공통 성분을 갖고 있는가?
- 반복적으로 등장하는 성분 중에서 나에게 알레르기나 자극을 유발할 가능성이 있는 성분은 무엇인가?

## Cosme_VISION이 지향하는 것

최종적으로는 사용자가 화장품을 `광고 기반`이 아니라 `성분 기반`으로 비교하고 선택할 수 있는 웹 플랫폼을 만드는 것이 목표입니다.

예를 들면 다음과 같은 흐름을 생각하고 있습니다.

- 미백, 자외선 차단, 진정 같은 기능 카테고리 안에서 제품을 검색한다.
- 제품별 전성분과 공시 정보를 비교한다.
- 내가 사용해 본 화장품을 기록한다.
- 그중 피부 트러블이 생겼던 제품을 체크한다.
- 문제가 있었던 제품들 사이의 공통 성분을 분석한다.
- 장기적으로는 나만의 피부 리스크 성분 프로필을 만든다.

즉, Cosme_VISION은 단순히 화장품을 나열하는 서비스가 아니라  
`내 피부 반응과 제품 성분을 연결해서 해석하는 분석 플랫폼`을 지향합니다.

## 첫 출발은 올리브영

프로젝트의 첫 시작은 가장 대표적인 화장품 판매 채널 중 하나인 `올리브영`에서 판매 중인 제품들을 기준으로 진행하려고 합니다.

현재는 다음 흐름을 기반으로 데이터를 수집하고 있습니다.

- `goodsNo` 수집
- `detail` API를 통한 상품 메타데이터 및 썸네일 확보
- `article` API를 통한 상품정보 제공고시 및 전성분 확보
- `description` API를 통한 상세 이미지 URL 보조 수집

이 단계는 전체 플랫폼을 만들기 위한 가장 첫 번째 데이터 기반 작업입니다.

## 현재 진행 방향

지금은 플랫폼의 완성형보다, 먼저 화장품 데이터를 안정적으로 수집하고 분석 가능한 구조로 만드는 데 집중하고 있습니다.

현재 주요 작업 방향은 아래와 같습니다.

- 올리브영 상품 메타데이터 수집
- 상품정보 제공고시 수집
- 전성분 텍스트 수집 및 정리
- 이미지 및 상세 리소스 정리
- 성분 비교를 위한 데이터 구조 설계
- 사용자별 트러블 유발 성분 패턴 분석 구조 설계

## 궁극적으로 만들고 싶은 것

Cosme_VISION이 궁극적으로 답하고 싶은 질문은 이런 것들입니다.

- 어떤 제품들이 실제 성분 구성이 비슷한가?
- 내가 트러블을 겪었던 화장품들에는 어떤 공통 성분이 있는가?
- 나는 앞으로 어떤 성분을 피하는 것이 좋을까?
- 내 피부 상태와 목적에 더 잘 맞는 제품은 무엇인가?

## 프로젝트 상태

이 프로젝트는 이제 막 첫 걸음을 뗀 단계입니다.

아직은 데이터 수집과 성분 분석 기반을 만드는 초기 단계이고, 앞으로 비교 로직, 사용자 맞춤 분석, Shiny 기반 웹 플랫폼 구조를 차근차근 확장해 나갈 예정입니다.

지금은 시작 단계이니, 앞으로 어떻게 발전해 가는지 지켜봐 주시면 좋겠습니다.

작은 첫 단계부터 시작하지만, 장기적으로는 신뢰할 수 있는 화장품 성분 분석 플랫폼으로 발전시키는 것이 목표입니다.
